# NeoKinokoLauncherVer2

## 共通項目
### エンコーダーについて
ソフト内で行う、ファイルの圧縮・解凍はshift-jisを利用
コード自体はUTF-8を利用

### githubに含まれていないファイルについて
権利上の理由や機密性の理由から一部のアセットが含まれていないため、一部の機能は正常に動作しません。
アセットはアセット群項目に書かれたアセットを導入した上で開発を行うようにしてください。(AssetStoreTools下に配置するか、自身でignoreに指定してください)
環境変数等、機密性の高いファイル群についてはDiscord等に別で配布するようにします。そちらをご確認ください。
